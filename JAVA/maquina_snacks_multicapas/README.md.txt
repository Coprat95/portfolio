
# 🥪 Multilayer Snack Machine

A Java console-based application that simulates a snack vending machine using a layered architecture. This project demonstrates object-oriented programming principles, modular design, and separation of concerns.

## 📦 Project Structure

The application is organized into three main layers:

- **domain/** → Core entities and business models (e.g., Snack, Slot)
- **service/** → Business logic and operations (e.g., vending, inventory management)
- **presentation/** → Console-based user interface and input/output handling

## 🎯 Features

- Display available snacks and prices
- Select and purchase snacks
- Manage inventory and refill slots
- Validate user input and handle errors
- Modular design for easy extension and testing

## 🛠️ Technologies Used

- Java SE 17
- Object-Oriented Programming (OOP)
- Layered architecture (Domain → Service → Presentation)
- Console-based UI
